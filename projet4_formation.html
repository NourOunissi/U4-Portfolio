Framework Laravel

Plus de détails

Laravel est un framework web open-source écrit en PHP, conçu pour simplifier et accélérer le développement d'applications web. Créé par Taylor Otwell, Laravel offre une structure élégante et expressive qui favorise la productivité des développeurs.

Pour ma part, j'ai travaillé avec ce framework afin de réaliser mon projet de client léger. Hera Bougie est un site web proposant la vente ou la création de bougie.
